# 환경 구성

## 실행 환경

- `nodejs` : 20.19.0
- `pnpm` : 19.8.1
- `nextjs` : 15.3.5

## 환경변수

```text
.env.production
NEXT_PUBLIC_API_SERVER_URL={API 서버 URL}
NEXT_PUBLIC_MOCK_SERVER_ENABLE=false
```

```text
.env.development
NEXT_PUBLIC_API_SERVER_URL=http://localhost:8080
NEXT_PUBLIC_MOCK_SERVER_ENABLE=true
```
